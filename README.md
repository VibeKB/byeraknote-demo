## 🎮 사용법

1. **오답 등록하기** 클릭
2. 이미지 선택 (최대 5MB)
3. 텍스트 입력 (정답, 오답 사유 등)
4. 저장
5. **오답노트 보기**에서:
   - 이미지 클릭 → 확대 보기
   - 정답보기 버튼 → 텍스트 표시/숨김
   - 이전/다음 버튼 → 노트 이동
   - 삭제 버튼 → 노트 삭제
